## How to Start the App
```
pnpm dev
```
Starts the Vite dev server for `@officexr/web` at http://localhost:5173.

For mobile (Expo):
```
pnpm dev:mobile
```

For desktop (Electron, after web build):
```
pnpm --filter @officexr/desktop run dev
```
